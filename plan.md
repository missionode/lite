# Chakra Meditation App - Project Plan (Master Edition)

## 1. Overview
A professional-grade, immersive web meditation app designed to induce deep trance and "ecstasy" states through traditional chakra wisdom and advanced psychoacoustic soundscapes.

## 2. Advanced Audio Engine (The "Ecstasy" Core)
- **Sub-Threshold Drone:** Static, blurry resonance at 0.03 gain (Subconscious support).
- **Procedural Singing Bowl:** Synthesized resonant bell for grounding (Wake-up & Transitions).
- **Fade-In/Out:** 5-second smooth transitions to prevent nervous system jars.
- **Spatial & Dynamic Audio:** 3D panning for circling sound and dynamic reverb spaces (cave to cathedral).
- **Elemental Layers:** Subtle procedural textures (Earth, Water, Fire, Air, Ether) per chakra.

## 3. Immersive Visual Engine
- **Chakra Breathing:** Ultra-subtle (2%) scaling of symbols for a "living" feel.
- **Aura Ambient Background:** Full-screen low-opacity color glow matching the active chakra.
- **Progress Journey:** 7-stage visual indicator showing the path from Root to Crown.
- **Canvas Particle System:** Mesmerizing, audio-reactive WebGL/Canvas particles.
- **AMOLED Black:** True black backgrounds to save battery and increase contrast.

## 4. UI/UX & Guidance
- **Box Breathing (Intro):** 1-minute physiological priming (4-4-4-4 technique).
- **Multi-Lingual Narration:** Ultra-slow (0.65 rate), deep-pitched guidance (English/Malayalam).
- **Native Typography:** Premium Malayalam web fonts for elegant localization.
- **Positive Affirmations:** "You Are" perspective seals the intent of each chakra.
- **Customizable Journeys:** Options to do full sequences or select individual/multiple chakras.
- **Volume Mixer:** Independent control over Narration, Drone, and Bell.
- **Meditation Stats:** Local streak and total meditation time tracking.

## 5. Technical Foundation
- **PWA / Offline:** Fully installable with Service Worker caching.
- **Wake Lock:** Prevents device sleep during the entire session.
- **Media Session API:** Lock screen and headphone hardware control integration.
- **Local Persistence:** Remembers user language, voice, and time preferences.
